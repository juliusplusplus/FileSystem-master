# FileSystem
操作系统: 二级目录文件系统 
 ------
    二级文件目录的实现
    用disk.txt模拟磁盘，使用Help查看支持的命令及其操作方式,root为超级用户(写在disk.txt中)
    
    文件的逻辑结构：流式文件。
    物理结构：链接文件。
    物理空间管理：空闲链法。
    目录结构：二级目录结构。
    目录搜索技术：线性搜索。
    FCB：含文件相关的全部属性。
